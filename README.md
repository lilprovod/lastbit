# LastBit
Алгоритм, вычисляющий скалярное произведение двух векторов с точностью до последнего значащего бита числа двойной точности IEEE 754 double.

*В разработке: сейчас реализовано сравнение наивного скалярного произведения, произведения с суммой Кэхэна и через алгоритма FMA*

# Инструкция по запуску

Для начала необходимо создать файл с входными данными в формате:
```
размерность векторов
вектор А (число 1, число 2, ...)
вектор Б (число 1, число 2, ...)
```

Например:
```
3
1.00000 2.00000 3.00000
3.00000 2.00000 1.00000
```

Сборка программы:
`make`

Запуск:
`./lastbit [filename]`
