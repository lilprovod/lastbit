# LastBit
Алгоритм, вычисляющий скалярное произведение двух векторов с точностью до последнего значащего бита числа двойной точности IEEE 754 double.

Доступные алгоритмы:

1. Сумма:
   - наивная сумма;
   - сумма Кэхэна;
   - сумма КБН второго и третьего порядков;
   - сумма Огита-Оиши;
   - **эталонная** сумма через длинную арифметику GMP.
2. Умножение:
   - наивное умножение;
   - **эталонное** умножение через FMA.
3. Скалярное произведение:
   - наивное («наивная сумма» + «наивное умножение»);
   - через Кэхэна («сумма Кэхэна» + «наивное умножение»);
   - через сумму КБН второго и третьего порядков («сумма КБН 2/3» + «наивное умножение»);
   - через сумму Огита-Оиши («сумма Огита-Оиши» + «наивное умножение»);
   - **эталонное** скалярное произведение («сумма через GMP» + «умножение через FMA»).

# Инструкция по запуску

Входные данные принимаются программой через файл в следующем формате:
```
размерность векторов
вектор А (число 1, число 2, ...)
вектор Б (число 1, число 2, ...)
```

Например:
```
3
1.00000 2.00000 3.00000
3.00000 2.00000 1.00000
```

Сборка программы происходит через `Makefile`. Доступные команды:
`make` – собрать проект;

`make clean` – очистить cwd от файлов сборки;

`make rebuild` – полностью пересобрать проект (применяется после изменения исходного кода).



Запуск в обычном режиме:
`./lastbit [filename]`



Запуск демонстрационного режима (примеры):

`./lastbit --demo`
